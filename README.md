## Getting Started

- This is Backend Part completely build on Php
- First Clone the main Frontend part from my Repo
- Then you copy all the folders cloned from here to [www] folder in wamp64 or whatever you have installed
- Then run the files using php files_name.php
- If you incounter package missing error then download the missing packages
- After that project should be able to run 100% 👍
